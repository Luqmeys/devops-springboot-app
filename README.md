# DevOps Spring Boot Application 🚀

This project is developed for **CSEB5143 DevOps: Tools and Technologies**.

It demonstrates a complete **CI/CD pipeline** for a simple Spring Boot REST API using:

- GitHub (Version Control)
- Maven (Build Automation)
- Jenkins (CI/CD Pipeline)
- Docker (Containerization)
- Kubernetes / Minikube (Deployment)

---

## 📌 Project Overview

This is a simple **Spring Boot REST API** application with basic endpoints for testing deployment and DevOps workflow.

### Available Endpoints
- `/` → Returns application welcome message
- `/health` → Returns application health status
